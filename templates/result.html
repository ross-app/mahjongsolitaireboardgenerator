Mahjong Solitaire Board Generator - Physical Tiles

This tool generates a solvable Mahjong Solitaire board layout designed for physical tiles.

The board is built using a turtle layout system, where each layer is stacked on top of the previous one. The first layer is always the bottom layer, and higher layers are placed above it.

This ensures that when you physically build and play the game, the board remains fully solvable.
